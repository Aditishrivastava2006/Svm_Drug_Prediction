# 💊 SVM Drug Prediction

This project predicts the most suitable drug for a patient using the Support Vector Machine (SVM) algorithm.

## 📌 Project Overview
This machine learning model predicts drug types based on patient information such as:
- Age
- Sex
- Blood Pressure
- Cholesterol
- Sodium to Potassium Ratio (Na_to_K)

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 🤖 Machine Learning Model
- Support Vector Machine (SVM)

## 📂 Dataset
- Drug Classification Dataset (Kaggle)

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
